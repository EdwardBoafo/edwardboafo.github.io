HTML Tags I've Learned

h1
The main heading tag used to define the most important heading on a page.
It ranges from h1 to h6, with h1 being the most important heading,
typically reserved for the main title or topic of the page.
p
The paragraph tag used to define blocks of text. It groups sentences together into paragraphs.
a
The anchor tag used to create hyperlinks to other pages or sections within a page.
img
The image tag used to embed images in a webpage. It requires the "src" attribute to specify the image source.
ul
The unordered list tag used to create bulleted lists. Items inside the list are defined using li tags.
